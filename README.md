##  Introduction
Modern desk lamps often provide fixed brightness and color, limiting adaptability to different environments. Lighting conditions significantly impact comfort, focus, and eye health.  

**LumiFlex** is a smart desk lamp featuring:  
- Adjustable brightness  
- Tunable warm-to-cool light  
- Wireless control via a mobile application  

By combining LED technology, a low-voltage power supply, and intuitive app-based control, LumiFlex provides a **customizable, user-friendly, and innovative lighting solution**.
